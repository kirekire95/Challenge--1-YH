## Inlämninguppgift 1 - Responsiv design med CSS Grid & Media Queries

- Skapa ett nytt projekt med minst 2 html filer som har
  minst 3 element varav ett av dessa element är en bild (img).

- Skapa en(alternativt flera) .css fil(er) och importera de html mall du tidigare skapat.

- I din .css skall du använda dig av CSS Grid & Media Queries för att ge elementen i projektet ett responsivt beteende.

- Man skall även kunna navigera sig mellan två olika vyer (exempelvis mellan en 'Home' och en 'About' vy)

Krav: Sidan skall vara responsiv till både Desktop samt Mobile och man skall kunna navigera sig mellan minst två olika vyer. varav båda skall vara responsiva.
